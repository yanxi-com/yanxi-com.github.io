### 安装 ruby 版本管理工具 rvm

```shell
# 安装 rvm
\curl -sSL https://get.rvm.io | bash -s stable

# rvm install 2.4.0
rvm install 2.4.0
```

### 通过 gem 安装

    gem install jekyll jekyll-paginate jekyll-gist

### 更新 gem 镜像

见 [http://gems.ruby-china.org/](http://gems.ruby-china.org/)

### 下载源代码 & 运行

```
git clone https://github.com/yanxi-me/h5nodejs.git
cd h5nodejs

jekyll s
```
